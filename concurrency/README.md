# 并发编程
1. 原子性
```
    x = 10;        // 赋值操作，直接将数值10写入到工作内存中，具备原子性
    y = x;         // 先读取x的值，再将x的值写入工作内存，两个动作有先后顺序，不再具备原子性
    x++;           // 读取x的值，进行加1操作，写入新的值，三个动作有先后顺序，不再具备原子性
    x = x + 1;     // 读取x的值，进行加1操作，写入新的值，三个动作有先后顺序，不再具备原子性
```
